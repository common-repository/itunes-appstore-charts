== Description ==

The iTunes Charts Widget is a free configurable widget to display the current download charts in all the Apple digital Stores. Choose how many items will be displayed

3, 5, 10, 25, 50, 75, 100, 200, 300

This Plugin is no longer just an Appstore Chart Widget, we have done a lot of work to implement all the digital Apple Stores like

* The iOS Appstore
* The Mac Appstore
* iTunes Music
* iTunes Video
* itunes Music Videos
* iTunes TV Shows

All these stores are listed with specific submmenues and selection options between free, paid and grossing items. All these configuration options can be reduced to specific genres.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `appstore_charts` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set widgets position in Design -> Widget
1. Set configuration under Settings -> iTunes Charts
